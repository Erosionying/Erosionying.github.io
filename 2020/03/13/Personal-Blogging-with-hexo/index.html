Personal Blogging with hexo

1、We need to download Node.js first,Please Click on https://nodejs.org/ to install,Select LTS Version to install.

note: The Version is not important

2、Check the installation using following Command , If Version number is printed , the Installation is Complete.

node --version

npm --version

3、Open Treminal, Execute the following Command installation and Checking hexo. If the following Output , Installation is Complete.

npm install -g hexo-cli

hexo --version

4、Create the new folder.Into the newly Created folder, Execute the following Command.

hexo init
