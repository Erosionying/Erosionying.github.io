IDEA中配置Visual Studio风格开发环境

Posted on 2020-11-13 | In IDE

Microsoft一直是我的信仰，即便我心开发Java代码，我心依旧留给.NET Core。正如，我身虽然使用IDEA，我心却早已给了Visual Studio。虽脚下山路难行，却依旧向往星空。太抒情了~ 我懂了，这就来波Visual Studio主题！

前几天吃着火锅唱着歌，刷着网络，发着呆，突然就找到了这么一个IDEA的开源主题，好家伙，原来有和我一样的人(老感动了)。源码Gitee链接: https://gitee.com/durcframework/VisualStudioThemeForIntelliJ。


选择任意一种方式下载，推荐直接点击Download ZIP(如果没有Gitee帐户，需要注册帐户)，使用Git或者SVN下载的都是源代码，需要打包，这里是想更加简单。

把压缩包最好自己去创建一个文件夹，这样有助于后期方便管理我们的文件(非强迫症玩家可以直接忽略)。

找到合适的地方，创建Theme文件夹，将下载的Zip包复制进去。

打开IDEA，打开File -> Manage IDE Settings -> Import Settings

找到自己下载的或自己压缩的Zip包，点击OK

完成这些之后，重启IDEA(避免玄学错误，这是个人习惯)。

再次打开IDEA，选择File -> Settings;

选择Editor -> Color Scheme 右侧Scheme下拉框中找到Visual Studio系列主题。

我使用的是Visual Studio的经典暗黑主题也就是Visual Studio 2019 Dark Optimize(带个Optimize的我觉着肯定要好一些)

选中点击右下角APPLY，然后点击OK，主题已经更换。
